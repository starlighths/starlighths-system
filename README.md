# 🌟 Starlight High School Management System (2025-2026)

## Project Overview
A cloud-based school management platform designed for the unique needs of Starlight High. Features include a 2-week alternating timetable, unified subject grading, and real-time extracurricular management.

### 🚀 Core Features
- **Smart Timetable:** 08:00–17:00 grid with automated cell merging and elective splitting.
- **Dual Class Teachers:** Integrated "Sync Slot" (15:45) for co-teacher coordination.
- **Unified English:** Weighted calculation of English Language + Speaking.
- **Grouped Subjects:** Cross-class rosters with individual student grade routing.
- **ECA Module:** Conflict-check logic for club applications.

### 🛠 Tech Stack Requirements
- **Database:** PostgreSQL (Relational)
- **Backend:** Node.js / Python
- **Frontend:** React.js (Responsive)
- **Auth:** JWT with Bcrypt Hashing

### 🔒 Admin Controls
- **Anytime Enrollment:** Add/Edit users mid-term.
- **Password Reset:** Admin-level overrides for locked accounts.
- **Audit Logs:** Full tracking of system modifications.
